# 新闻分析系统

这是完整的新闻获取与分析系统。用户只需使用主入口 `news-analyzer`
根据完整工作流程制定调用计划，系统会自动完成所有步骤。

## 技能分类

### 用户技能

| 技能 | 命令 | 功能 |
|------|------|------|
| 新闻分析系统 | `news-analyzer` | **主入口** - 输入关键词或URL，执行完整分析流程 |

### 内部技能（由系统自动调用）

| 技能 | 命令                            | 功能 |
|------|-------------------------------|------|
| 联网查询 | `news-search` | 联网搜索相关内容，获取2-3条结果 |
| 新闻爬取 | `news-crawler` | 爬取完整的新闻内容，包含原文图片 |
| 文章详情 | `news-detail` | 根据爬取内容获取新闻的完整详情 |
| 新闻真实性验证 | `news-verification` | 验证新闻真实性，识别虚假信息 |
| 新闻时间线梳理 | `news-timeline` | 梳理相关新闻的时间线 |
| 新闻趋势预测 | `news-trend-prediction` | 预测新闻事件的发展趋势 |
| 输出报告 | `news-report` | 汇总分析结果，生成结构化报告 |
