# 新闻爬取

这是新闻分析系统的内部工具技能，用于爬取并转换新闻内容。
只进行3个网页内容的解析
## 功能

1. **获取页面**
   - 接收新闻链接URL
   - 访问新闻页面，提取完整内容

2. **内容转换**
   - 将HTML格式转换为Markdown格式
   - 保留原文结构（标题、段落、列表等）
   - 保留原文中的所有图片以及相应位置（使用图片链接）

3. **提取信息**
   - 标题
   - 作者/来源
   - 发布时间
   - 原文链接url

4. **输出格式**
```json
{
  "title": "新闻标题",
  "author": "作者",
  "source": "来源媒体",
  "published_time": "发布时间",
  "url": "原文链接也就是link",
  "content": "## 新闻标题\\n\\n正文内容（Markdown格式）\\n\\n![图片描述](图片URL)\\n\\n...",
}
```
