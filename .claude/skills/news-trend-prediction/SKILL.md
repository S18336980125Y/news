# 新闻趋势预测

请对新闻事件进行趋势预测分析：

1. **现状分析**
   - 梳理当前事件状态
   - 识别关键参与者和利益相关方

2. **因素识别**
   - 分析影响事件走向的关键因素
   - 识别正面/负面影响力量

3. **情景推演**
   - 乐观情景（最好情况）
   - 基准情景（最可能情况）
   - 悲观情景（最坏情况）

4. **输出报告**
   - 短期预测（1-7天）
   - 中期预测（1-4周）
   - 关键观察指标
   - 需要关注的后续信号
