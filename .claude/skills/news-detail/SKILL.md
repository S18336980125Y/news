# 文章详情

请生成文章的完整详情报告，包含以下模块：

## 1. 基础信息
- 标题
- 作者/来源
- 发布时间
- 摘要（100-200字）
- 原文链接（真是的联网查询出的link的url）

## 2. 正文内容
- 完整正文（保留原文格式）
- 完整原文图片（使用md![图片描述](图片URL)的形式）
- 图片必须是原文内容中的图片，禁止使用其他新闻下的图片


## 输出格式
建议使用 Markdown 格式输出，便于阅读和分享。
不要丢掉网页解析出来的原文图片链接；
