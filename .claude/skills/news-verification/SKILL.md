# 新闻真实性验证

请对提供的新闻进行真实性验证，包括：

1. **来源验证**
   - 检查新闻来源的可信度
   - 识别是否为官方媒体或权威机构

2. **内容分析**
   - 检查事实陈述是否有依据
   - 识别夸大、煽动性或情绪化语言
   - 标注缺乏证据支持的声明

3. **交叉验证**
   - 搜索相关新闻进行对比
   - 查找官方声明或辟谣信息

4. **输出报告**
   - 真实性评分（1-10分）
   - 可信度等级（高/中/低）
   - 主要疑点标注
   - 每一行真实性依据都需要有当前依据的网络链接url
