# 联网查询

这是新闻分析系统的内部工具技能，用于联网搜索相关内容，只允许查询中国国内的新闻媒体。
如果是带有时间的今日，今天这种需要获取当前日期作为具体搜索条件，比如2026年X月X日。

## 功能

1. **接收查询**
   - 接收搜索关键词
   - 例如："人工智能最新进展"、"某公司产品发布"

2. **联网搜索**
   - 使用网络搜索工具
   - 以中国国内各大新闻媒体为主要搜索源
   - 新闻网站如新浪网，东方财经网、搜狐网、腾讯网、网易、新浪财经网
   - 获取相关内容链接

3. **筛选结果**
   - 筛选最新、最相关的不超过2条
   - 优先选择权威媒体来源
   - 去除重复或低质量内容


# 网页搜索

使用 MCP搜索网页内容。

## 输出格式

将搜索结果以清晰的格式展示给用户，包括：
- 标题
- 链接
- 摘要/内容预览
- 日期
- 来源评分
